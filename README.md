# Masonry Grid (React)

## Install

```bash
  npm i @k-art/masorny
  yarn add @k-art/masorny
```

## Usage

```tsx
  import { MasonryGrid } from '@k-art/masonry';

  <MasonryGrid items={[React.FC, React.FC, ..., React.FC]} gap={24} />
```
